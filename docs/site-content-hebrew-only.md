# תוכן האתר — עברית בלבד (לפי סדר הדף)

קובץ **מקור לעריכה**: עדכן כאן את הניסוח, החזר ליישום ב־`lib/translations.ts` (שדות `he`) או ברכיבים (טבלאות/מספרים קשיחים). אחר כך אפשר לייצר גרסה אנגלית מקבילה.

**סנכרון:** התוכן משקף את המאגר הנוכחי — `lib/translations.ts`, `app/page.tsx` והרכיבים שבסוף המסמך.

**שפת ברירת מחדל בטעינה:** אנגלית; עברית אחרי מתג השפה (`localStorage` — `eqs-lang`).

**מעל הגלילה (לא חלק מ־`<main>`):** `Navbar` קבוע למעלה + **רקע גלובלי** (רשת, אוניות, נקודות צפות — ללא טקסט). סדר **לחיצות** בתפריט — ב**נספח ב׳** בסוף המסמך (לא סדר גלילה).
