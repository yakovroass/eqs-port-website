# נשאר רק מה שאני לא יכול לעשות בשבילך

אלה הפעולות היחידות שדורשות **התחברות לחשבון שלך** – אין לי גישה.

---

## 1. AWS Amplify (בדפדפן שנפתח)

1. **התחבר** ל-AWS (או צור חשבון).
2. **Create new app** → **Host web app**.
3. בחר **GitHub** וחבר אם מתבקש.
4. בחר רפו **eqs-port-website**, ענף **main**.
5. **Save and deploy** – Amplify יבנה מהקוד ב-GitHub (כולל `amplify.yml`).
6. חכה ל-build ירוק. תקבל כתובת כמו `https://main.xxxxx.amplifyapp.com`.

---

## 2. Hostinger (אחרי ש-Amplify עלה)

1. ב-**Amplify** → האפליקציה → **Hosting** → **Custom domains** → **Add domain**.
2. הזן את הדומיין: **eqsport.io**
3. **בחר "Manual configuration"** (לא "Create hosted zone on Route 53") – כך זה עובד עם חשבון Free Tier ועם דומיין ב-Hostinger.
4. **Configure domain** – Amplify יציג אילו רשומות DNS להוסיף. **העתק** אותן (CNAME וכו').
5. **Hostinger** → הדומיין eqsport.io → **DNS / DNS Zone** → הוסף את הרשומות **בדיוק** כמו ש-Amplify ביקש.
6. שמור. אחרי כמה דקות (עד 24 שעות) הדומיין יפנה לאתר.
