# איך להפעיל את טופס "צור קשר" (שליחת הודעות למייל)

כשאתה שולח הודעה מהטופס ואתה רואה את השגיאה:
**"Site email isn't set up yet. Add WEB3FORMS_ACCESS_KEY in Amplify..."**
זה אומר שהאתר עדיין לא מחובר לשירות שליחת מיילים. כך מתקנים:

---

## שלב 1 — קבלת מפתח חינם מ-Web3Forms

1. גלוש ל־**https://web3forms.com**
2. הזן את **כתובת המייל** שלך (לשם יגיעו ההודעות)
3. לחץ **"Create Access Key"** (או "Get Access Key")
4. **העתק** את המפתח שנוצר (מחרוזת ארוכה)

---

## שלב 2 — הוספת המפתח ב-AWS Amplify

1. היכנס ל־**AWS Amplify Console**: https://console.aws.amazon.com/amplify
2. בחר את **האפליקציה** של האתר (eqs-port-website)
3. בתפריט הצד: **Hosting** → **Environment variables**  
   (או: **App settings** → **Environment variables**)
4. לחץ **Edit** / **Manage variables**
5. **הוסף משתנה חדש:**
   - **Name:** `WEB3FORMS_ACCESS_KEY`
   - **Value:** המפתח שהעתקת מ-Web3Forms
6. **שמור** (Save)

---

## שלב 3 — פריסה מחדש (Redeploy)

אחרי שמירת המשתנה, Amplify לא תמיד מפעילה build אוטומטי.

- אם מופיעה אפשרות **"Redeploy this version"** — לחץ עליה  
- אחרת: עשה **שינוי קטן** בקוד (למשל הוסף רווח ב־README), העלה ל-Git (commit + push) — ואז Amplify תבנה ותפרוס מחדש אוטומטית

---

## בדיקה

אחרי ה-Redeploy, היכנס לאתר, מלא את טופס "צור קשר" ולחץ **Send Message**.  
ההודעה אמורה להישלח ולהגיע לכתובת המייל שהזנת ב-Web3Forms.

---

## אם עדיין לא עובד

- ודא שהמשתנה נקרא **בדיוק** `WEB3FORMS_ACCESS_KEY` (כתיב אנגלי, עם קו תחתון)
- ודא שהעתקת את **כל** המפתח מ-Web3Forms בלי רווחים מיותרים
- ב-Web3Forms: בדוק ב-Dashboard שלא חסום (חינם עד 250 שליחות בחודש)
