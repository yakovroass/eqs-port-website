# תוכן האתר — עברית בלבד (לפי סדר הדף)

קובץ **מקור לעריכה**: עדכן כאן את הניסוח, החזר ליישום ב־`lib/translations.ts` (שדות `he`) או ברכיבים (טבלאות/מספרים קשיחים ב־`MarketData.tsx`, `FutureTargets.tsx` וכו׳). אחר כך אפשר לייצר גרסה אנגלית מקבילה.

**סנכרון:** עודכן לפי המאגר הנוכחי — `lib/translations.ts`, `app/page.tsx`, `components/Navbar.tsx`, `Hero.tsx`, `MarketData.tsx`, `Problem.tsx`, `ProcessRealityIntro.tsx`, `SolutionOverview.tsx`, `ProcessTimeline.tsx`, `FutureTargets.tsx`, `Investment.tsx`, `Contact.tsx`.

**שפת ברירת מחדל בטעינה:** אנגלית; עברית אחרי מתג השפה (`localStorage` — `eqs-lang`).

**מעל הגלילה (לא חלק מ־`<main>`):** `Navbar` קבוע למעלה + **רקע גלובלי** (רשת, אוניות, נקודות צפות — ללא טקסט).

### סדר גלילה ב־`<main>` (`app/page.tsx`) — כך מופיע התוכן מלמעלה למטה

1. `#hero` — Hero  
2. `#market` — השוק (`MarketData`)  
3. `#problem` — הבעיה (`Problem`)  
4. `#process` — רכש גלובלי / מבוא תהליך (`ProcessRealityIntro`)  
5. `#solution` — פתרון (`SolutionOverview`)  
6. `#process-detail` — פירוט מסכים / ציר תכולה (`ProcessTimeline`; **אין קישור בתפריט**)  
7. `#future` — מפת דרכים (`FutureTargets`)  
8. `#invest` — השקעה (`Investment`)  
9. `#contact` — צור קשר + פוטר (`Contact`)

**תפריט עליון:** סדר העוגנים **תואם** לסדר הגלילה. ל־`#process` הטקסט בתפריט הוא שילוב `realityHeadlineLead` + `realityHeadlineAccent` (בעברית: **רכש גלובלי היום**). ל־`#future` הטקסט הוא `roadmap.label` (**מפת דרכים עתידית**), לא `nav.future`.

להלן **אותו סדר בדיוק**: כל כותרת `##` היא שלב בגלילה, והתת־סעיפים בתוכה לפי סדר ה־DOM ברכיב.
