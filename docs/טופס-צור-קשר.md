# טופס צור קשר – שליחה למייל

הטופס שולח דרך השרת. **מומלץ: Web3Forms** (בלי אימות דומיין).

## אפשרות א׳ — Web3Forms (הכי פשוט)

1. היכנס ל־**https://web3forms.com** → Get Access Key (חינם).
2. ב־**AWS Amplify** → Environment variables → הוסף:
   - `WEB3FORMS_ACCESS_KEY` = המפתח שהעתקת
3. **Redeploy** את האפליקציה.

ההודעות יגיעו לכתובת שתגדיר ב-Web3Forms (Dashboard).

## אפשרות ב׳ — Resend

1. **https://resend.com** → API Key.
2. ב־Amplify: `RESEND_API_KEY`, ואופציונלי `CONTACT_EMAIL`, `RESEND_FROM`.
3. Redeploy.

אם מוגג גם **WEB3FORMS_ACCESS_KEY** וגם Resend — ייעשה שימוש ב־Web3Forms קודם.

## אם לא הוגדר שום מפתח

יוצגת הודעה שהשירות לא מוגדר; **לא** ייפתח Gmail אוטומטית.
