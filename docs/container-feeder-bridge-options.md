# מגדל פיקוח — `container-feeder-top.svg` (אונייה 11)

**לראות את כל האפשרויות בדפדפן:** פתח בפרויקט את הדף **`/ship-bridge-demos`** (אחרי `npm run dev`).

בקובץ האנייה יש **4 אפשרויות** במבט על; בכל רגע רק אחת צריכה להיות **גלויה** — לכן בגלריה (`/ship-gallery`) תמיד תופיע רק אחת על הספינה.

## איך לבחור

1. פתח את `public/ship-refs/container-feeder-top.svg`.
2. חפש את הקבוצות:
   - `id="bridge-feeder-opt-1"` … `opt-4`
3. על האפשרות שתרצה **להציג**: מחק את `style="display: none"` (או שנה ל־`display: block`) — ל־**אפשרות 1** אין כרגע `display:none`.
4. על כל השאר: הוסף `style="display: none"` על תג ה־`<g>` הפתיחה.

## תיאור קצר

| # | שם | תיאור |
|---|-----|--------|
| **1** | מקונן קלאסי | שלושה מלבנים מקוננים, קווי חלון במישור, מכ״ם — עיגול + טבעת מקווקוות במרכז. **ברירת מחדל.** |
| **2** | משושה | קונטור משושה + משושה פנימי, שני קווים אופקיים, מכ״ם כעיגול מלא. |

| **4** | מדרגה | בלוק תחתון רחב + בלוק עליון צר יותר (מבט על “מדרגה”), שני קווים, מכ״ם. |
