## איפה רואים שינויים "מיד" (לפני שהעולם רואה)

### אפשרות א': `npm run dev` (מומלץ לעבודה)

בטרמינל, מתוך תיקיית הפרויקט:

```text
npm run dev
```

נפתח כתובת כמו **http://localhost:3000**

- **כאן** אתה רואה את האתר עם השינויים **מיד** (או אחרי רענון דפדפן).
- **רק אתה** במחשב שלך – אף אחד אחר לא רואה את זה.
- זה המקום לנסות רעיונות, לשחק עם טקסטים, לבדוק לפני פרסום.

### אפשרות ב': לפתוח קבצים בדפדפן (HTML סטטי)

חלק מהקבצים (אם יש) אפשר לפתוח ישירות – פחות רלוונטי ל-Next.js המלא.

**סיכום:**  
**שינוי מיידי = localhost (אחרי `npm run dev`)**, לא הדומיין.

## מתי השינויים מגיעים לאתר האמיתי (הדומיין)

המסלול הוא:

1. **שמירה** בקבצים במחשב (כבר קרה בעריכה).
2. **העלאה ל-GitHub** – `git add` → `git commit` → `git push` לענף `main`.
3. **AWS Amplify** מזהה push חדש, מריץ **build** אוטומטי, ומעדכן את האתר.

רק אחרי שלב 3 **המבקרים ב-www.eqsport.io** רואים את הגרסה החדשה.

---

## כמה זמן עד שהעדכון מופיע בדומיין?

| שלב | זמן משוער |
|-----|-----------|
| `git push` ל-GitHub | שניות |
| Amplify מתחיל build | כמה שניות עד דקה |
| Build של Next.js (ב-Amplify) | בדרך כלל **3–10 דקות** (תלוי בעומס) |
| CDN / מטמון דפדפן | לפעמים צריך **רענון חזק** (Ctrl+Shift+R) או דקה נוספת |

**בפועל:** אחרי שאמרת "תעלה" ועשינו push – תצפה ל־**בערך 5–15 דקות** עד שהאתר החי יראה כמו אצלך ב-localhost (אם ה-build הצליח).

## עבודה בלי לפרסם עד שאתה אומר

מה שאתה רוצה:

- לחשוב, לשנות, לנסות – **בלי** לעדכן את האתר החי בכל פעם.
- רק כשאתה מרוצה – **אז** לפרסם.

**איך זה עובד בפועל:**

1. **בזמן עבודה:** עורכים רק במחשב + `npm run dev` לבדיקה. **לא** עושים `git push` (או לא עושים commit בכלל אם רוצים לשמור רק מקומית).
2. **כשאתה מוכן לפרסם:** אתה אומר במפורש משהו כמו: **"תעלה את העדכונים לאתר"** / **"Let's publish"** – ואז:
   - commit + push ל-GitHub
   - Amplify יבנה ויעדכן את **www.eqsport.io**

כל עוד **לא** עשית push ל-`main`, **הדומיין לא משתנה**.

**הערה:** אם מישהו אחר (או אתה בטעות) דוחף ל-`main`, Amplify **כן** יפרסם אוטומטית. אם תרצה שליטה מלאה, אפשר בעתיד לעבוד עם ענף נפרד (למשל `develop`) ולמזג ל-`main` רק בפרסום – אפשר להגדיר זאת כשתרצה.
