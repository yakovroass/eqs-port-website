# טופס צור קשר – שליחת הודעות למייל

הטופס שולח הודעות ישירות למייל שלך דרך **Resend** (במקום לפתוח את תוכנת המייל של המבקר).

## מה לעשות פעם אחת

1. היכנס ל־**https://resend.com** וצור חשבון (חינם, 100 מיילים ביום).
2. צור **API Key** ב־Resend (API Keys → Create).
3. ב־**AWS Amplify** → האפליקציה → **Environment variables**:
   - הוסף `RESEND_API_KEY` עם הערך של המפתח שהעתקת.
   - אופציונלי: `CONTACT_EMAIL` אם אתה רוצה לקבל לכתובת אחרת (ברירת מחדל: yakovroass@gmail.com).
4. **Redeploy** את האפליקציה (או חכה ל־deploy הבא אחרי push).

אחרי זה, כל שליחה מטופס "צור קשר" תגיע למייל שלך.

## אם לא הגדרת RESEND_API_KEY

הטופס ינסה לשלוח; אם השירות לא מוגדר, האתר יפתח אוטומטית את תוכנת המייל של המבקר (ההתנהגות הישנה), כך שהטופס עדיין שמיש.
